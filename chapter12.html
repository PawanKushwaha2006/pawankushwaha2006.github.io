Chapter 12: HTML Tables (Basic)

Introduction to HTML Tables

HTML tables are used to display data in rows and columns. They are perfect for showing structured data like price lists, schedules, or comparisons.

Basic Table Tags

  • <table> - Defines the table container
  • <tr> - Table Row - creates a new row
  • <th> - Table Header - bold, centered heading cell
  • <td> - Table Data - regular cell with data

1. Creating a Simple Table

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>City</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
    <td>New York</td>
  </tr>
  <tr>
    <td>Sarah</td>
    <td>30</td>
    <td>London</td>
  </tr>
</table>

Output:

Name Age state
Pawan 22 Bihar
Vivek 18 UP

2. Table Structure Explained

<table> <!-- Start table -->
  <tr> <!-- Row 1 - Headers -->
    <th>Column 1</th> <!-- Header cell -->
    <th>Column 2</th>
  </tr>
  <tr> <!-- Row 2 - Data -->
    <td>Data 1</td> <!-- Data cell -->
    <td>Data 2</td>
  </tr>
</table> <!-- End table -->

3. Adding Borders with CSS

The border="1" attribute is old. Use CSS for modern styling:

<style>
  table {
    border-collapse: collapse;
    width: 100%;
  }
  th, td {
    border: 1px solid #333;
    padding: 8px;
    text-align: left;
  }
  th {
    background-color: #0ea5e9;
    color: white;
  }
</style>

<table>
  <tr>
    <th>Product</th>
    <th>Price</th>
  </tr>
  <tr>
    <td>Laptop</td>
    <td>\$999</td>
  </tr>
</table>

Output:

Product Price
Laptop $999
Mouse $25
Keyboard $75

  </tr>

5. Important Rules

✅ Best Practices:
1. Always use <th> for headers - improves accessibility
2. Keep number of cells same in each row
3. Use border-collapse: collapse to remove double borders
4. Add padding to cells for better readability
5. Don't use tables for page layout - use CSS Grid/Flexbox instead

6. Common Mistakes ❌

<!-- Wrong: Missing <tr> tag -->
<table>
  <th>Name</th> <!-- Should be inside <tr> -->
</table>

<!-- Wrong: Unequal columns -->
<tr>
  <td>Data 1</td>
  <td>Data 2</td>
</tr>
<tr>
  <td>Data 3</td> <!-- Missing second cell -->
</tr>
