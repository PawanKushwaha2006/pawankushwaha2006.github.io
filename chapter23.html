Chapter 23: Head Metadata for SEO

Meta tags go inside <head>. Users don't see them, but Google and browsers do. They control how your page appears in search results, social shares, and browser tabs.

1. Essential Meta Tags - Must Have

1. Title Tag - Most Important for SEO

<title>Best HTML Course for Beginners 2026 | LearnWebDev</title>
RuleDetails
Length50-60 characters. Google cuts off after ~600px
UniqueEvery page needs unique title
KeywordsPut main keyword near start
BrandAdd brand at end: " | SiteName"
No stuffingDon't repeat keywords: "HTML HTML HTML Course"
Google Search Result Preview:
https://learnwebdev.com/html-course
Best HTML Course for Beginners 2026 | LearnWebDev
Master HTML from scratch with 30 chapters, live examples, and projects. Perfect for beginners. Start coding today with our free course.
Google Can Rewrite: If your title is bad or keyword-stuffed, Google will auto-generate one from H1 or content. Write for humans first.

2. Meta Description

<meta name="description" content="Learn HTML from zero to hero with 30 detailed chapters, live code examples, and real projects. Free course updated for 2026.">
RuleDetails
Length150-160 characters. Google shows ~155
UniqueEach page needs unique description
Call to ActionInclude: Learn, Buy, Download, Start
KeywordsInclude main keyword naturally
No quotesAvoid " double quotes - breaks HTML
Not a Ranking Factor: Google said in 2009 that meta description doesn't affect rankings. But it affects CTR - click-through rate. Better description = more clicks = better rankings indirectly.

3. Viewport Tag - Mobile SEO

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Critical: Without this, Google marks your site "Not Mobile-Friendly" and ranks you lower on mobile. Required since 2015.

4. Charset Declaration

<meta charset="UTF-8">

Must be first tag in <head>. Ensures emojis, Hindi, Chinese render correctly. Prevents text encoding issues.

2. Robots Meta Tags - Control Indexing

Tells Google whether to index page or follow links.

<!-- Allow indexing - Default, optional -->
<meta name="robots" content="index, follow">

<!-- Block from Google completely -->
<meta name="robots" content="noindex, nofollow">

<!-- Index but don't follow links -->
<meta name="robots" content="index, nofollow">

<!-- Don't index but follow links -->
<meta name="robots" content="noindex, follow">
ValueMeaningUse When
indexShow in Google searchNormal pages
noindexHide from GoogleAdmin pages, thank-you pages, duplicate content
followCrawl links on pageDefault
nofollowDon't crawl linksUser-generated content, untrusted links
noarchiveDon't show cached versionPrice pages, stock data
nosnippetDon't show description in resultsLogin pages

Google-Specific Robots:

<!-- Only for Googlebot -->
<meta name="googlebot" content="noindex">

<!-- Control snippet length -->
<meta name="robots" content="max-snippet:150">

<!-- Control video preview -->
<meta name="robots" content="max-video-preview:30">

<!-- Control image size -->
<meta name="robots" content="max-image-preview:large">

3. Canonical Tag - Fix Duplicate Content

Tells Google which URL is the "main" version if you have duplicate/similar pages.

<link rel="canonical" href="https://example.com/html-course/">
When to Use Canonical:
Example: These URLs show same content:
1. https://example.com/course
2. https://example.com/course?ref=twitter
3. https://example.com/course/

Add <link rel="canonical" href="https://example.com/course"> to all 3. Google will index only #1 and pass all SEO value to it.

4. Open Graph Tags - Social Media

Controls how your page looks when shared on Facebook, LinkedIn, WhatsApp, Discord.

<!-- Open Graph / Facebook -->
<meta property="og:title" content="Best HTML Course 2026">
<meta property="og:description" content="Learn HTML with 30 chapters and projects">
<meta property="og:image" content="https://example.com/og-image.jpg">
<meta property="og:url" content="https://example.com/html-course">
<meta property="og:type" content="website">
<meta property="og:site_name" content="LearnWebDev">
TagRecommendedNotes
og:image1200x630px, <8MBMost important for shares
og:title60 charsCan differ from <title>
og:description200 charsCan differ from meta description
og:typewebsite, article, productarticle for blog posts

5. Twitter Card Tags

Controls how your page looks when shared on Twitter/X.

<!-- Twitter Card -->
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:site" content="@YourTwitterHandle">
<meta name="twitter:title" content="Best HTML Course 2026">
<meta name="twitter:description" content="Learn HTML with 30 chapters">
<meta name="twitter:image" content="https://example.com/twitter-image.jpg">
Card Types: summary = small image, summary_large_image = big image 1200x628px. Use large for better CTR.

6. Other Important Meta Tags

Language Tag:

<html lang="en">
<meta http-equiv="content-language" content="en-us">

Author & Copyright:

<meta name="author" content="John Doe">
<meta name="copyright" content="LearnWebDev 2026">

Refresh/Redirect:

<!-- Redirect after 5 seconds - Not recommended for SEO -->
<meta http-equiv="refresh" content="5; url=https://example.com/new-page">
Avoid: meta keywords tag. Google ignored it since 2009. Doesn't help SEO, wastes space.

7. Complete SEO Head Template

Copy this for every page. Replace values.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  
  <!-- Primary SEO -->

  <meta name="description" content="Master HTML from scratch with 30 chapters, live examples, and real projects. Free course updated for 2026. Start coding today.">

  <meta name="robots" content="index, follow">

  <!-- Open Graph / Facebook -->
  <meta property="og:title" content="Best HTML Course for Beginners 2026">
  <meta property="og:description" content="Learn HTML with 30 chapters and real projects. Free course.">
  <meta property="og:image" content="https://learnwebdev.com/images/html-course-og.jpg">

  <meta property="og:site_name" content="LearnWebDev">
  
  <!-- Twitter Card -->
  <meta name="twitter:card" content="summary_large_image">
  <meta name="twitter:site" content="@LearnWebDev">

  1. Google Rich Results Test: search.google.com/test/rich-results
  2. Facebook Debugger: developers.facebook.com/tools/debug
  3. Twitter Card Validator: cards-dev.twitter.com/validator
  4. LinkedIn Inspector: linkedin.com/post-inspector
  5. View Source: Right-click page → View Page Source → Check <head>

10. Summary - Golden Rules

  1. Title: 50-60 chars, unique, keyword first, brand last. Most important tag
  2. Description: 150-160 chars, unique, include CTA. Affects CTR not ranking
  3. Viewport: Mandatory for mobile SEO. Copy-paste the standard one
  4. Charset: Always UTF-8 and first tag in head
  5. Canonical: Fix duplicate content. Use absolute URLs
  6. Robots: Default is index,follow. Use noindex for private pages
  7. OG Tags: Required for good social shares. Image is most important
  8. Twitter Cards: Use summary_large_image for best results
  9. No Keywords Meta: Delete it. Google ignores since 2009
  10. Test: Use Facebook/Twitter debuggers before sharing
Pro Tip: Your <title> and meta description are your ad copy on Google. Write them to get clicks, not just for keywords. Higher CTR = better rankings over time.
