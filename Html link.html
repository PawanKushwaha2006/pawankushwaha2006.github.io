Chapter 7: HTML Links (Anchors)

The <a> tag creates hyperlinks. It lets users click and jump to another page, section, file, or website.

1. Basic Syntax

<a href="URL">Clickable Text</a>

href = "hypertext reference" - tells where to go.
Text between tags = What user sees and clicks.

Example:
<a href="https://www.google.com">Visit Google</a>

Output: Visit Google

2. External Links

Link to a different website/domain. Always use full URL with https://

<a href="https://www.w3schools.com">W3Schools</a>
<a href="https://github.com">GitHub</a>
Key Point: Without https://, browser thinks it's an internal file and gives 404 error.

3. Internal Links

Link to another page or section within your own website.

A. Link to Another Page:

<a href="about.html">About Us</a>
<a href="contact.html">Contact Page</a>
<a href="/blog/post1.html">Read Blog</a>

B. Link to Section on Same Page - Jump Links:

Step 1: Give target an id

<h2 id="services">Our Services</h2>

Step 2: Link to it using #

<a href="#services">Jump to Services</a>

4. The target Attribute

Controls where link opens.

ValueBehaviorUse Case
_selfOpens in same tab. DefaultInternal pages
_blankOpens in new tab/windowExternal links, PDFs
_parentOpens in parent frameFor iframes
_topOpens in full body windowFor iframes

Most Important: target="_blank"

<a href="https://www.google.com" target="_blank">Google</a>
Why Use It? Keeps your website open while user checks external site. User doesn't lose your page.
Security Best Practice: Always add rel="noopener noreferrer" with _blank 
<a href="https://external-site.com" target="_blank" rel="noopener noreferrer">
  Visit External Site
</a>

noopener = Prevents new page from controlling your page
noreferrer = Hides referrer info for privacy

5. Other Useful Link Types

A. Email Link:

<a href="mailto:support@example.com">Email Us</a>

Output: Email Us - Opens user's email app

B. Phone Link:

<a href="tel:+919876543210">Call Us</a>

Output: Call Us - On mobile, starts dialer

C. Download Link:

<a href="files/resume.pdf" download>Download Resume</a>

Forces file download instead of opening in browser.

D. Link to Top of Page:

<a href="#">Back to Top</a>

# alone jumps to top of current page.

7. Common Mistakes

MistakeWrongCorrect
Forgetting https://<a href="google.com"><a href="https://google.com">
No target for external<a href="https://site.com"><a href="https://site.com" target="_blank">
Missing rel attribute<a target="_blank"><a target="_blank" rel="noopener">
Empty href<a href=""><a href="#"> or <a href="/page">
Bad link text<a>Click Here</a><a>Read HTML Tutorial</a>

8. Summary - Golden Rules

  1. Internal Links: href="page.html" or href="#section" - No target needed
  2. External Links: href="https://..." + target="_blank" + rel="noopener noreferrer"
  3. Always use descriptive text: <a>Click Here</a> is bad. <a>Read HTML Tutorial</a> is good for SEO
  4. Test your links: Broken links hurt user experience and SEO
  5. Use # for same-page jumps with id attributes
